# Agentic Code-Intelligence Platform

A production-oriented backend service that allows users to submit
natural-language engineering tasks against large repositories.

The platform provides:

• Structural code retrieval
• Budget-aware agent execution
• Patch-based code editing
• Isolated sandbox verification
• Replayable task execution
• Multi-user workspace isolation
• Cost and budget enforcement

The system is designed for repositories that exceed model context
windows and therefore relies on structural navigation rather than
whole-repository prompting.
